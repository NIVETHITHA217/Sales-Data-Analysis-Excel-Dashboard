# Sales Data Analysis & Dashboard (Excel)
This beginner-level Excel project analyzes 2025 sales data using PivotTables, charts, and slicers to create an interactive dashboard.

### Key Insights:
- Total and Monthly Sales Analysis
- Top Products by Sales
- Interactive filters by Region and Category

### Tools Used:
- Microsoft Excel (PivotTables, Charts, Slicers)
- Data Visualization
